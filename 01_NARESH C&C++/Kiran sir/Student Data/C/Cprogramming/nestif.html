Example:14 

 main() 
 { 
  float a=0.7; 
  if(a<0.7) 
  printf("Stoned");
  else 
  printf("Averaged"); 
 } 

OUTPUT:  
          Stoned. 

Explanation: 
	 The output is very surprising! 0.7
is never less than 0.7, so the condition 
should evaluate to FALSE.But that doesn't 
happen, Reason is ,when 0.7 is stored in a,
due to precision considerations, it is 
stored as something less than 0.7.naturally,
when value stored in a is compared with 0.7,
the condition evaluates to TRUE and Stoned 
gets printed.

Two get rid of the problem there are 
2 solutions. 

(a) declare A as a long Double as 
     shown below: 

      long double a; 
(b) Typecast 0.7 to a Float while 
    comparing as shown below: 
 
      if(a<(float)0.7) 

Typecasting means converting to the 
specified data type.
