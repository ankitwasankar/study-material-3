More Examples on Logical Operators



Example:24 

             main()
             { 
              int i=4,z=12; 
              if(i==4||z>50) 
              printf("Dean of Student affairs"); 
              else 
              printf("Dosa"); 
             } 

OUTPUT:   Dean of Student affairs 

Explanation:  Logical OR(||)  operator in the IF evaluates
 to TRUE.That's why the output results.
