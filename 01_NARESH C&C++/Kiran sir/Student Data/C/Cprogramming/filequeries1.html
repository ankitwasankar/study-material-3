2.Point out the error ,if any in the following program:

# include "stdio.h"
main( )
{ unsigned char;  
  FILE *fp;
  fp=fopen("ieg",r);
  while((ch=getc(fp))!=EOF)
  printf("%c",ch);
fclose(fp);
}

Solution:EOF has been defined as #define EOF -1 in the
file "stdio.h" and an unsigned char ranges from the file
it cannot be accomodated in ch.Therefore solution is to
declare ch as int.

9.point out the error in the following program:
 # include "stdio.h"
  main( )
  {
    char ch;
    int i;
    scanf("%c",&c);
    scanf("%d",&ch); 
    printf("%c%d",ch,i);
  }

solution: You would not get achance to supply a character for
the second scanf( ) statement .solution is to preceede the 
second scanf( ) with the following statement fflush(stdin);
  This would flush out the enter hit for the previous scanf( )
to be flushed out from the inputstream ,i.e., keyboard.
