Which To Use When


 1.A variable is declared as static if one wants the value
of the variable to be persists between different function
calls.

2.A variable is declared as register storage class if those
variables are used very often in program.

3.A variable is declared as external, when that variable is
being used by almost all functions in the program.

4.Except all above conditions, we can declare variables as
 automatic. However default variable declaration  of 
  variable as auto.
