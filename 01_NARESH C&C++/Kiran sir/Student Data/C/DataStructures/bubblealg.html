Algorithm :

Procedure bubblesort(A,N)
1.For pass=1 to N-1 do
begin
2.NoExchange <- true
3.For j=1 to (N-pass) do
begin
4.If a[j]>a[j+1] then
begin
5.swap(a[j],a[j+1])
6.NoExchange <- False
end
end
7.If(NoExchange is true) then
return
end
8.Return.

                 The Bubble sort is simple but least efficient. If the data set contains N elements,the maximum number of passes is N-1. Similarly the number of passes is N-1. Similarly the number of tests per pass ranges between N-1 and 1 with an average value of N/2. Thus the upper bound on the total number of tests is roughly (N-1)*N/2 that is proportional to N2.
