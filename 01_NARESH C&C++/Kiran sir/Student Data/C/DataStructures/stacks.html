Why use ADT's?
     
    Precisely specify the nature of the data and the operations
ADT's are independent of representation (indeed, of programming
language). No matter what language we will be using a complex 
number data type means the same thing. Provide solutions to common
problems, giving us a toolkit we can use to solve problems 
("building blocks"). Many ADT's are concerned with how values of 
some other type can be stored and accessed. Classes which implement
such storage types are called "container classes" in C++ programming.
